Privacy & Security

Fae processes everything locally by default. Your voice, conversations, and memory stay on your device.

Local Processing

Speech recognition (STT), text-to-speech (TTS), and LLM inference all run on-device using optimised models. No audio or text is sent to external servers unless you explicitly configure an API provider.

Permissions

Microphone: Required for voice input. Audio is processed locally and never stored permanently.

Contacts: Optional. Used to personalise greetings by reading your name from the "Me" card. Contact data is never uploaded.

Calendar: Optional. Enables proactive briefings about upcoming events. Data stays local.

Memory

Fae stores conversation memory in ~/.fae/memory/. This data is encrypted at rest using macOS Keychain and never leaves your device.

Network Access

Fae only connects to the internet for:

• Downloading voice models on first launch

• Checking for app updates

• API calls if you explicitly configure a cloud LLM provider

Data Deletion

To completely remove all Fae data, delete ~/.fae/. Permissions can be revoked in System Settings > Privacy & Security.
